# Assets

## monica.jpg

Coloque a foto oficial da Monica Buonfiglio aqui com o nome **`monica.jpg`**.

- **Formato recomendado:** JPG quadrado, mínimo 800×800px
- **Onde aparece:** tela "Sobre a Monica" no app (`/?action=monica` ou via card no Home)
- **Fallback:** se o arquivo não existir, o app mostra um placeholder com as iniciais "MB" automaticamente — sem quebrar nada.

## Como subir

1. Pegue a foto enviada (ou pegue uma alta-resolução do perfil oficial)
2. Renomeie para `monica.jpg`
3. Coloque nesta pasta `/sob-as-asas/assets/monica.jpg`
4. Faça commit + push → Vercel deploya em ~1 min
5. Abra `https://www.sobasasas.com.br/assets/monica.jpg` no navegador pra confirmar que carrega
